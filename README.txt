Car_Black_Box_Project

First Step to upload the .hex file in the MPLAB.IDE

STEP BY STPE PROCEDURE WHEN THE PROJECT IS RUNNING:
Login_Password : 1111

1. Dashboard screen
	SW1 - Collision
	SW2 - GEAR UP level
	SW3 - GEAR DOWN level
	SW4 or SW5 - Login Screen

2. Login screen
	SW4 - Enter '1'
	SW5 - Enter '0'

3. Main Menu screen
	SW4 - Up the menu
	SW5 - Down the menu
	LPSW4 - Long press to select
	LPSW5 - Long press to Logout

4. View_log
	SW4 - increment the log view
	SW5 - decrement the log view
5. Reset_Time
	SW4 - increment the current time filed
	SW5 - toggle the time field (HH:MM:SS)
	SW6 - Store the Time to RTC register

6. Change_Password 
	SW4 - Enter '1'
	SW5 - Enter '0'
